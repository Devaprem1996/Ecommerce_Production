## Rules
- Do NOT show technical error details to users
- Log error details to monitoring service
- Show support contact clearly
- "Try Again" button: retry the last action
- Auto-retry after 30s (optional with countdown)

## Error Boundary
- This file wraps all pages
- Catches unexpected React errors too
- Prop: error (Error object) + reset (retry function)
