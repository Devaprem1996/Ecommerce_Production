# ADMIN — CONTENT MANAGEMENT (CMS)

## URL: /admin/content

## Purpose
Enables store managers to update dynamic text assets, promotional homepage banners, blogs, and FAQ questions without editing source code.

## Layout & Management Tabs

### TAB 1: BANNER MANAGER
- Grid listing current homepage carousel slides.
- Actions:
  * Reorder via drag-and-drop.
  * `[+ Add Slide]`: Form containing Title (English/Tamil), Subtitle, Background Image Upload (max 2MB, optimized for landscape), CTA Button text, and CTA Link URL.
  * Delete or toggle slide visibility.

### TAB 2: BLOG MANAGER
- Table listing all blog articles with columns: Thumbnail, Title, Date Published, Status (Published/Draft/Scheduled).
- Editor: Integrated TipTap rich text editor.
- Metadata: Author profile, SEO Meta description, and search tag labels.

### TAB 3: FAQ ACCORDION MANAGER
- List grouped by categories (e.g. `Delivery`, `Payment`, `Products`).
- Form:
  * Category dropdown.
  * Question Input * (Bilingual English/Tamil).
  * Rich-text Answer TextArea *.

### TAB 4: ACCREDITATION & BADGES
- Manager for logos and certifications displayed on the homepage (e.g. `USDA Organic`, `FSSAI Approved`).
- Fields: Brand Logo, Label text, detail descriptions.
