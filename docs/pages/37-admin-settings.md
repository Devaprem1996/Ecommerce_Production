# ADMIN — SITE SETTINGS

## URL: /admin/settings
## Access: Super Admin only

## Settings Sections (Tabs)

### TAB 1: GENERAL
  Site Name *
  Site Tagline
  Site Logo (upload)
  Favicon (upload)
  Contact Email *
  Contact Phone *
  WhatsApp Number *
  Address (full business address)
  Google Maps Embed URL
  Business Hours

### TAB 2: PAYMENT SETTINGS
  Payment Gateway: [Razorpay / PayU / Stripe]
  API Key (masked — show/hide)
  API Secret (masked — show/hide)
  Webhook URL (auto-generated)
  Test Mode: [On / Off] toggle
  Payment Methods:
    ☑ UPI
    ☑ Credit/Debit Card
    ☑ Net Banking
    ☑ Cash on Delivery
    ☑ Wallets (Paytm, PhonePe)
  COD Extra Charge: ₹__

### TAB 3: DELIVERY SETTINGS
  Free Delivery Threshold: ₹__
  Default Delivery Charge: ₹__
  Same-day Cutoff Time: [2:00 PM]
  Order Processing Days: [Mon-Sat]
  Holidays (add dates to exclude)

### TAB 4: EMAIL SETTINGS
  SMTP Host, Port, Username, Password
  From Email, From Name
  Test Email button
  Email Templates link (goes to template manager)

### TAB 5: SMS SETTINGS
  SMS Provider: [Twilio / MSG91 / Textlocal]
  API Key
  Sender ID
  Test SMS button

### TAB 6: SEO SETTINGS
  Default Meta Title
  Default Meta Description
  Google Analytics ID
  Facebook Pixel ID
  Google Search Console verification
  Robots.txt editor
  Sitemap URL (auto-generated)

### TAB 7: SOCIAL MEDIA
  Facebook URL
  Instagram URL
  Twitter/X URL
  YouTube URL
  Pinterest URL
  LinkedIn URL

### TAB 8: MAINTENANCE
  Maintenance Mode: [On / Off] toggle
  Maintenance Message
  Expected Back Time
  Allowed IPs (admin bypass)

### All Settings:
  [Save Settings] button per tab
  Success toast on save
  Unsaved changes warning
