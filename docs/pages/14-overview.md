# OVERVIEW PAGE

## URL: /overview

## Purpose
Give first-time visitors a quick snapshot of who we are,
what we sell, and why to trust us. 
Bridge between Home and About pages.

## Sections

### 1. PAGE HERO
- Simple gradient background (Primary Green)
- Heading: "What We're All About"
- Subheading: One-liner brand mission
- No CTA buttons (just informational)
- Height: 40vh

### 2. BRAND SNAPSHOT (Stats Section)
- 4 animated count-up numbers:
  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
  │  500+    │ │  10,000+ │ │  50+     │ │  5       │
  │ Products │ │ Customers│ │ Farmers  │ │ Years    │
  └──────────┘ └──────────┘ └──────────┘ └──────────┘
- Trigger count-up when scrolled into view
- Animation duration: 2 seconds each

### 3. OUR MISSION
- Two column layout (desktop) / stacked (mobile)
- Left : Illustration (unDraw/Storyset SVG)
- Right: Mission heading + paragraph text
- Highlight key words in Primary Green

### 4. WHAT WE OFFER (Category Overview)
- Grid of category cards (same as homepage categories)
- Each: icon + name + short description + item count
- "Browse Category →" link on each
- 2 columns mobile / 3 columns tablet / 4 columns desktop

### 5. HOW IT WORKS
- 4-step process (horizontal on desktop, vertical on mobile)
  Step 1: 🔍 Browse Products
  Step 2: 🛒 Add to Cart
  Step 3: 📦 We Pack Fresh
  Step 4: 🚛 Delivered to You
- Connected with dotted line (desktop)
- Animation: steps reveal left to right on scroll

### 6. OUR VALUES
- 3 large value cards:
  🌿 100% Organic → We source only certified organic
  👨🌾 Direct Farmers → We work directly with 50+ farmers
  🔬 Lab Tested → Every batch tested for purity
- Cards with hover lift animation
- Icon + Title + Description

### 7. MINI TESTIMONIALS
- 3 short testimonial quotes (not full carousel)
- Simple text quotes with star rating
- Link: "Read All Reviews →"

### 8. CTA SECTION
- Full-width green background
- Heading: "Ready to Eat Healthier?"
- Button: [Shop Now →]

## Design Notes
- PAGE is informational — no product cards needed
- Clean reading experience
- Animations: AOS fade-up on each section
- Mobile: single column throughout
