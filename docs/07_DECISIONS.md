# Architecture & Design Decisions (ADRs)

Record of key decisions made in the project.

## Decisions

- **ADR-001: Project Directory Layout**
  - **Status**: Approved
  - **Context**: Need a clear separation of frontend, backend, deployment, database, assets, and documentation.
  - **Decision**: Created the folder structure containing `frontend/`, `backend/`, `docs/`, `database/`, `deployment/`, `scripts/`, `assets/`.
