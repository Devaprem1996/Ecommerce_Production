# Ecommerce Production Project

Full stack development with cloud pipelines. This is the main repository for the ecommerce production project.

## Directory Structure

- `docs/` - Project documentation, planning, and guidelines.
- `backend/` - Backend services and API source code.
- `frontend/` - Frontend application source code.
- `database/` - Database schemas, migrations, and scripts.
- `deployment/` - Infrastructure as Code, Docker configs, and deployment scripts.
- `scripts/` - Utility scripts.
- `assets/` - Project assets (images, icons, etc.).
